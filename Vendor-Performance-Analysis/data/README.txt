DATASET INFORMATION

The original sales and purchases datasets used in this project were not uploaded to GitHub due to file size limitations.

Project analysis was performed using:

* Sales Dataset
* Purchases Dataset

The datasets contained millions of transaction records and were used for:

* Vendor revenue analysis
* Vendor profit analysis
* Profit margin analysis
* Pareto (80/20) analysis
* Vendor lead time analysis
* Power BI dashboard creation

All analysis code, SQL queries, business insights, and visualizations are available in this repository.
